# Webcam_NativeApp
Create a simple GUI program in Pyqt5 for the task of detecting objects on the screen in real time using a webcam.


## Start
```
git clone https://github.com/ultralytics/yolov5.git
```

```
pip install -r requirements.txt
```

```
python webapp.py
```
